# Get All Comments

All Comments of all Users.

**URL** : `/api/comments`

**Method** : `GET`

**Auth required** : None

## Success Response

**Code** : `200 OK`

**Content examples**


```json
[
    {
        "_id": "5c9956b241a32752d242973a",
        "username": "test1",
        "postId": "5c99473c6027ee44c6b15a76",
        "comment": "This is a nice picture",
        "__v": 0
    }
]
```
